About EV Station Finder USA&CAN

EV Station Finder USA&CAN is an Android application for USA and Canada, which calculates a route and shows electric loading stations with electric vehicle supply equipment (EVSE), along this route with a given maximum distance to the route or within a circle with a given radius around a location. The location may be obtained by the location service of the phone or an address entered by the user.
The EVSE stations along the route can be restricted to the vicinity of refuelling stops, whose distance from each other is the cruising range with the achievable loading state of batteries at refueling stations, which often is lower than 100% of the capacity of the batteries.

The To Address of a route request is now the final end address without waypoints separated by semikolon. Waypoints can be added in arbitrary order to a list. If stations are already found, a selected station can be added to the list. The next route request shows the loop way from the route to this station.

In Settings of the action bar the user can configure the start action, which is shown at start of the app. The start action can be the Route Request or the Circular Search of charging stations. The start action may be shown or executed immendiately applying the last used search criteria.

The following navigation systems can be used to navigate to a selected point of the route or a selected EVSE station:

  • Google Maps
  • Sygic GPS
  • Navigon GPS
  • WAZE GPS
  • HERE WeGo
  • OsmAnd
The list of stations can be saved to shared external storage. It can be imported to navigate to a selected station. This is done without using the internet.

There is a free version EV Station Finder USA&CAN and a paid version EV Station Finder USA&CAN Plus.
EV Station Finder USA&CAN Plus provides additional functions for maintaining station data.
Maintaining consists of downloading station data from a web site to external storage (as sdcard), updating from external storage and resetting to the initially installed station files.

Setting the language

EV Station Finder USA&CAN is bilingual. If the language of the phone was set in 'Settings', 'Language & Keyboard' or 'Language & Input'to German, all texts of EV Station Finder USA&CAN appear in German, for all other languages in English.
If the language was set to English(UK) or English(US), distances are measured in miles.
If the language is modified, while EV Station Finder USA&CAN is active, then generated lists as route and station details are shown in the previous language as long as these lists are generated again.

User views

The user interacts with EV Station Finder USA&CAN by user views.
There are the following user views:
  • The application starts with the view Map and shows an empty map centred at the current location. After a route was calculated and after loading stations are present, the stations are shown on the map.
  • View Route contains a form to request a new route or shows a formerly calculated route.
  • View Stations searches for EVSE stations and shows the station details.
  • Plus version only: view Maintain shows a form for downloading station data from the internet to external storage (e.g. sd card), for updating station data from external storage and for resetting data to the originally installed station data.
  • With view Import a list of stations, which was stored on external storage, can be imported.
Navigating to other views is done by option menus. The menu items are shown in the action bar on top of the screen. Dependent on available space an item is shown with its icon and with title. If the space is not sufficient, the icons of the first items are shown; the remaining items are shown by clicking the vertical three dots.

Clicking the left icon (3 horizontal lines or an arrow to the left) in the action bar at the top of the screen toggles between the views Route, Map and Stations. The same can be done by swiping from the left side of the screen or back or by swiping from the right side of the screen or back.
The views Route and Stations cover a part of the map, which is slightly dimmed.

Clicking the back key immediately finishes a view or dialog and returns to the view, which initially invoked it. The data of the finished view are lost.

View Route

For requesting a route enter the address of the start location into the 'From' field and the address of the final destination into 'To' field.(Since V1.0.4 without waypoints separated by semicolon ';').

Button Waypoints opens a list of waypoints. A waypoint can be added as an address. If stations are already found, a station can be added as a waypoint by selecting it on the map and adding it in the view Waypoints or by entering the station number.
Adding a charging station as waypoint will show the loop way from the route to the station after the next route request.

By clicking the black cross on orange background a waypoint can be removed. The order of waypoints is arbitrary.
By clicking the button Ready the actual waypoints will be applied in requesting a route.
Addresses may be in the form used by Google Maps, for example:
  • complete address:Zip code, City, street house number, country name or country code
    example: 4056, Basel, Hebelstrasse 11, CH
  • incomplete address: City or City, street
  • geographical latitude, longitude in decimal degrees
    example: 43.0488, -76.1429
If the the From field is long clicked, the last known position of the phone is used as the From address. Google's Location Service must be activated in the phone's Settings.

Buttons:

  • Swap From/To Address:
    Start address and final destination address are swapped.
  • Avoid Highways: if checked highways will be avoided.
  • Get Route: get the route.

Menu items in the action bar: s. View Map

When the result of the route request is available the route details are shown and the route is displayed on the map.
After a successful route request the settings of the request are saved and restored after restart of EV Station Finder USA.

View Map

The view Map presents the requested route and the searched stations on the map.

Use of Google Maps

EV Station Finder USA&CAN is based on Google Maps, which requires that Google Play Services are installed on the device. If not, the user is requested to install these services from Google Play Store. This requires that the app Google Play Store is installed and the device is signed on with its GoogleMail account.

Menu items

The actionbar on top of the screen contains the menu items. It can be accessed by the views Route, Map and Stations.
  • Settings: regarding connector types and charging levels of stations to be searched and map views (satellite, traffic info and current location).
  • Stations: toggles between the views Stations and Map.
  • Import: opens a file picker for selecting a formerly saved station list for importing from external storage.
  • Maintain (Plus version only): opens the sub nenu for maintaining stations data.
  • Help: opens help information.

Handling of touch/click events

The route is shown as a red line together with green markers at start, stopovers and end point. The stations are presented as markers for higher zoom levels. The color of the marker indicates the charging level of the station: violet for level 1, blue for level2, yellow for DC fast and light green for legacy charging systems.
For lower zoom levels stations are clustered; the cluster markers show the number of contained stations.

Clicking a cluster marker zooms in and shows the station markers of this cluster.

Clicking a marker will open an info window with a description of this marker. A short click on the map closes the info window.

Clicking an arrow button in the middle left and right of the screen steps to the next marker, forwards by the the right button and backwards by the left button. This affects route and station markers.

Using a navigation sytem

If a route marker or a station marker is selected, clicking the button with a blue right arrow opens a dialog asking the user, if he wants to navigate with the shown navi system from the current position to the selected marker. The address of the selected marker is shown, if an internet connection exists. Clicking the field with the navi system opens a list of navi systems, from which one may be selected. Clicking Yes opens the selected navigation system, if it is installed on the android device.

The following navigation systems may be used:
  • Google Maps Navigation
  • Sygic GPS Navigation with version >12.0
  • Navigon with version > 3.5.2
  • WAZE GPS Navigation
  • Nokia HERE Navigation

If the navigation system was finished, view Map will be opened. If the android device was running on low memory, EV Station Finder USA&CAN may be finished and will start from scratch.

View Stations

With the view Stations EVSE stations can be searched along a route or around a location within a given radius. The stations along a route may be restricted to the vicinity of refuelling stops, which are apart from the next stop approximately in a distance of the refuelling range.

The radius search can be done around a location given by the location service of the phone or determined by an address enetered by the user.

The result of searching for stations can be saved to a html file on external storage (e.g. sd card). This file can be imported by clicking the menu item Import. This can be used for navigating to a station of this file without having an internet connection.

The station view contains two subviews:
  • Search criteria for searching stations.
  • Station list with the search results.
Each sub view may be switched to the other sub view.

Menu items: s. View Map

Search Criteria

  • Combo box Search Mode:
    Clicking the combo box opens a list with possible search modes
    • Around Location: search around a location obtained by the location service of the phone within a circle of the given radius.
      When the search mode Around location is selected and no location service was activated, it is asked, if the settings Location access should be opened or the last known location should be used.
    • Around Address: search around a location given by the address entered by the user within a circle of the given radius.
      When this mode is already selected, the address of the location can be modified by long clicking the combo box.
      When the search mode Around address is selected and no network was activated, the settings of the phone are opened.
    • Along Route: search along the route with a miximum distance Corridor from the route.
    • Refueling Stops: as mode Along Route restricted to the vicinity of refueling stops. At end of the route a circular search is done to charge for the way back.
  • Corridor: distance of the route in kms or miles for English (UK/US).
  • Ranges for search mode Refueling Stops:
    • Range after refueling stops in km or miles: range with achievable loading state at refueling stations; this is often lower than the battery capacity.
    • Start Range in km or miles: range at start of a trip; this can be greater than the Range after refueling stops.
      This takes into account, that a trip can be started with fully loaded batteries, which often cannot be fully reloaded at refueling stops.
  • Button Find stations: starts searching.
  • Button Station List: switches to sub view Station List.
The station data required for the selected search mode will be loaded, after searching for stations was started, if they are not already loaded. Only then the result of a former search will be overridden.
The states of all modifiable fields are saved after a search was completed successfully and restored after EV Station Finder USA&CAN was restarted.

Searching stations for the area of a route or location the first time, will load the station data corresponding to the settings of Connectors and Charge Levels. The number of loaded stations is shown in the first line.

Station List

The station list shows the stations with the following informations:
  • Name
  • Address
  • Date of last confirmation
  • Phone number
  • Connector Type
  • Number of loadings station of supported types
  • Name of the electric network
  • Opening hours
  • Distance from start of the route or from user's location and for searching along the route the airline distance from the route.
  • Geographical latitude and longitude in degrees.
  • Hyperlink to the electric network
There are the following buttons or hyperlinks:
  • Button Top:
    scrolls to the top of the station list.
  • Button Save:
    Saves the station list as a html file in the shared external storage. Later on this file can be imported or accessed by other Apps as file managers or email.
  • Button Search Criteria:
    switches to the sub view Search Criteria.
  • List of stations:
    Clicking the blue hyper link of a station selects the corresponding station marker on the map and opens an info window. This becomes completely visible, after the view Stations was hidden.
    If a station was selected on the map, swichting to view Stations the view is scrolled to the details of this station.

View Import

In view Import a file picker is shown to select a station list file in the shared external memory. This file can be accessed by other Apps as file manager or email.

This can be used advantageously, if no internet connection is available or roaming fees should be avoided. A station list is prepared in a Wlan environment and stored at shared xternal storage. While traveling this list is imported to navigate to a selected station.

Menu items:
  • Get file: opens the file picker for selecting a station lst file.
  • Top: scrolls to the top of the list.

View Maintain

(Available in EV Finder Plus only)
The station data are onboard of EV Station Finder USA. Each version will be shipped with actual station data. Station date are frequently modified. So the installed data of EV Station Finder USA&CAN are soon out of date. Updating station dta is reqired to use actual data
:The station data come from the following web site

https://developer.nrel.gov/


Maintaining station data means:
  • Downloading station data:

    The station data of USA and Canada are downloaded from a web site to external storage (as sdcard). Downloading is done by a background service, which runs independently on the app EV Station Finder USA&CAN Plus. The end of downloading is notified in the status bar at the top of the screen. Swiping down will show the results of the downlaod. After downloading was finished successfully, station data will be updated from sdcard.

    The user can decide to load station data for ´USA and Canada or only for one of these countries.

    The download service can be started explicitly by the user or can run periodicly. If the periodic mode is selected, at start of the app the service is started, when the period has elepased after the last download.

    Periodic downloading:

    Checking the checkbox Periodic activates the periodic mode. The period in weeks is set in the field Weeks.

    Explicit start:

    Clicking the button Start will start the download service.
  • Updating station data from sdcard:

    If the external storage contains station data in a specific directory structure, clicking the button 'update station data ..' will replace the actual station data by those on sdcard.

  • Resetting to initial station data:

    If it turns out that the updated station data are incorrect, the user can reset to the initially installed station data.

Note:

The author of EV Station Finder USA&CAN takes no warranty that the downloaded station data are correct, and takes no responsibility about damages resulting by using incorrect data.
