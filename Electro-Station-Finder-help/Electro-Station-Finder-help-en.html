About Electro Station Finder EUR

Electro Station Finder calculates a route and searches for electric charging stations. Charging stations can be searched along this route or around a location.
Charging stations along the route can be restricted to the vicinity of refuelling stops, whose distance from each other is the cruising range with the achievable loading state of batteries at refueling stations, which often is lower than 100% of the capacity of the batteries.

In Settings of the action bar the user can configure the start action, which is shown at start of the app. The start action can be the Route Request or the Circular Search of charging stations. The start action may be shown or executed immendiately applying the last used search criteria.

The route is shown on the map as a red line with route markes at start, intermediate destinations and final destination. Charging stations are presented as cluster or single station markers dependent on the zoom level.
A navigation system can be selected for navigating from the current position to the selected marker.

The list of stations can be saved to sd card. It can be imported to navigate to a selected station. This is done without using the internet.

There are a free version Electro Station Finder EUR and a paid version Electro Station Finder Plus EUR. The Plus version offers additional search criteria Free Charging, Barrier Free Payment system and Charge Cards.

The station data are provided by https://www.goingelectric.de/
Electro Station Finder downloads the required station data for each new route request and for each new location.

GoingElectric provides and maintains station data nearly for all European countries. Completeness and actuality depends on registered users, who can send updates of the stations. This works pretty well for Austria, Switzerland and Germany. Other countries may lack of completeness and actuality.

Setting the language

Electro Station Finder is bilingual. If the language of the phone was set in 'Settings', 'Language & Keyboard' or 'Language & Input'to German, all texts of Electro Station Finder appear in German, for all other languages in English.
If the language was set to English(UK) or English(US), distances are measured in miles.
If the language is modified, while Electro Station Finder is active, then generated lists of a route and stations are shown in the previous language as long as these lists are generated again.

User views

The user interacts with Electro Station Finder by user views.
There are the following user views:
  • The application starts with the view Map and shows an empty map centred at the current location, it this is known by the phone After a route was calculated and after loading stations are present, the stations are shown on the map.
  • View Route contains a form to request a new route and shows a formerly calculated route.
  • View Stations searches for charging stations and shows a short description of the station.
  • With view Import a list of stations, which was stored on external storage, can be imported.
Navigating to other views is done by option menus. The menu items are shown in the action bar on top of the screen. Dependent on available space an item is shown with its icon and with title. If the space is not sufficient, the icons of the first items are shown; the remaining items are shown by clicking the vertical three dots.

Clicking the left icon (3 horizontal lines or an arrow to the left) in the action bar at the top of the screen toggles between the views Route, Map and Stations. The same can be done by swiping from the left side of the screen or back or by swiping from the right side of the screen or back.
The views Route and Stations cover a part of the map, which is slightly dimmed.

Clicking the back key immediately finishes a view or dialog and returns to the view, which initially invoked it. The data of the finished view are lost.

View Route

For requesting a route enter the address of the start location into the 'From' field and the address of the final destination into 'To' field.(Since V1.0.4 without waypoints separated by semicolon ';').

Button Waypoints opens a list of waypoints. A waypoint can be added as an address. If charging stations are already found, a station can be added as a waypoint by selecting it on the map and adding it in the view Waypoints or by entering the station number.
Adding a charging station as waypoint will show the loop way from the route to the station after the next route request.

By clicking the black cross on orange background a waypoint can be removed. The order of waypoints is arbitrary.
By clicking the button Ready the actual waypoints will be applied in requesting a route.
Addresses may be in the form used by Google Maps, for example:
  • complete address:Zip code, City, street house number, country name or country code
    example: 4056, Basel, Hebelstrasse 11, CH
  • incomplete address: City or City, street
  • geographical latitude, longitude in decimal degrees
    example: 49.0488, 11.1429
If the the From field is long clicked, the last known position of the phone is used as the From address. Google's Location Service must be activated in the phone's Settings.

Buttons:

  • Button Waypoints: opens the view for editing waypoints.
  • Swap From/To Address:
    Start address and final destination address are swapped.
  • Avoid Highways: if checked highways will be avoided.
  • Get Route: get the route.

Menu items in the action bar: s. View Map

When the result of the route request is available the route steps are shown in a list and the route is displayed on the map.
After a successful route request the settings of the request are saved and restored after restart of Electro Station Finder.

View Map

The view Map presents the requested route and the searched stations on the map.

Use of Google Maps

Electro Station Finder is based on Google Maps, which requires that Google Play Services are installed on the device. If not, the user is requested to install these services from Google Play Store. This requires that the app Google Play Store is installed and the device is signed on with its GoogleMail account.

The actionbar on top of the screen contains the menu items. It can be accessed by the views Route, Map and Stations.
  • Settings:
    • Selection of satellite view, showing traffic info and current location button.
    • Selection of the start action Routen Request or Circular Search..
    • With/Without immediate execution of the start action.
  • Stations: toggles between the views Stations and Map.
  • Import: opens a view for selecting a formerly saved station list for importing from external storage.
  • Help: opens help information.

Handling of touch/click events

The route is shown as a red line together with green markers at start,stopovers and end point. Stations are presented as markers for higher zoom levels. The color of the marker indicates the maximum power of the charging points of this station:
violet < 11 KW (L=1), blue < 22 KW (L2), yellow < 50 KW, brown >= 50 KW (L4).
For lower zoom levels stations are clustered; the cluster markers show the number of contained stations.

Clicking a cluster marker zooms in and shows the station markers of this cluster.

Clicking a marker will open an info window with a short description of this marker. A short click on the map closes the info window.
Clicking the bar Get Station Details of an info window opens the details of this charging station.

Clicking an arrow button in the middle left and right of the screen steps to the next marker, forwards by the the right button and backwards by the left button. This affects route and station markers.

Using a navigation sytem

If a route marker or a station marker is selected, clicking the button with a blue right arrow at the bottom of the screen opens a dialog asking the user, if he wants to navigate with the shown navi system from the current position to the selected marker. The address of the selected marker is shown, if an internet connection exists. Clicking the field with the navi system opens a list of navi systems, from which one may be selected. Clicking Yes opens the selected navigation system, if it is installed on the android device.

The following navigation systems may be used:
  • Google Maps
  • Sygic GPS
  • Navigon
  • WAZE GPS
  • HERE WeGo
  • OsmAnd

If the navigation system was finished, view Map will be opened. If the android device was running on low memory, Electro Station Finder may be finished and will start from scratch.

View Stations

With the view Stations charging stations can be searched along a route or around a location within a given radius. The stations along a route may be restricted to the vicinity of refuelling stops, which are apart from the next stop by the distance of the refuelling range.

The radius search can be done around a location given by the location service of the phone or determined by an address enetered by the user.

The result of searching for stations can be saved to a html file on external storage (e.g. sd card). This file can be imported by clicking the menu item Import. This can be used for navigating to a station of this file without having an internet connection.

The station view contains two subviews:
  • Search criteria for searching stations.
  • Station list with the search results.
Each sub view may be switched to the other sub view.

Menu items: s. View Map

Search Criteria

  • Combobox Connector:
    Clicking the combo box opens a list of availabe connector types. Any number of connectors can be checked including none. Searching for charging stations results in those stations, which have at least one of the checked connectors with the required loading power. If no connector is checked, searching is independent on connectors and loading power.
  • Combobox Power:
    Clicking the combo box opens a list of available minimal charging powers of charge points. Searching for stations results in those stations, which have a charge point with one of the selected connectors and its charging power greater or equal the selected value.
  • Corridor: distance of the route in kms or miles for English (GB/US).
  • Combo box Search Mode:
    Clicking the combo box opens a list with possible search modes
    • Around Location: search around a location obtained by the location service of the phone within a circle of the given radius.
      When the search mode Around location is selected and no location service was activated, it is asked, if the settings Location access should be opened or the last known location should be used.
    • Around Address: search around a location given by the address entered by the user within a circle of the given radius.
      When this mode is already selected, the address of the location can be modified by long clicking the combo box.
      When the search mode Around address is selected and no network was activated, the settings of the phone are opened.
    • Along Route: search along the route with a miximum distance Corridor from the route.
    • At charging Stops: as mode Along Route but restricted to the vicinity of charging stops. At end of the route a circular search is done to charge for the way back.
  • Ranges:
    • Range after charging stops in km or miles: range with achievable loading state at charging stations; this is often lower than the battery capacity (for mode Charging Stops only).
    • Start Range in km or miles: range at start of a trip; this can be greater than the Range after charging stops.
      This takes into account, that a trip can be started with fully loaded batteries, which often cannot be fully reloaded at charging stops.
  • Button Find stations: starts searching.
  • Button Station List: switches to sub view Station List.

Plus Version only:

  • Free Charging: results in charging stations, which allow charging without costs.
  • Barrier Free Payment: results in charging stations, which can be used with normal payment methods as cash, EC-. credit- or RFID-cards.
    Hint: The search criteria Barrier Free Payment and Charge Cards are linked by Or.
  • Charge Cards: A multiple choice list with available charge cards is presented with a search field on top. The search field can be filled with the prefix of charge card names. The list scrols down to this prefix.
    The selected charge cards are linked by Or.
    The accepted charge cards of a station can be seen in the details of a station.
The station data required for the selected search mode will be loaded, after searching for stations was started, if they are not already loaded. Only then the result of a former search will be overridden.
The states of all modifiable fields are saved after a search was completed successfully and restored after Electro Station Finder was restarted.

Station List

The station list shows the stations with the following informations:
  • Name
  • Address
  • Connector Types with number of available charging points and power in KW
  • Distance from start of the route or from user's location and for searching along the route the airline distance from the route.
  • Geographical latitude and longitude in degrees.
  • Hyperlink to the charging station
There are the following buttons or hyperlinks:
  • Button Top:
    scrolls to the top of the station list.
  • Button Save:
    Saves the station list in a html file on the external storage (e.g. sd card).
  • Button Search Criteria:
    switches to the sub view Search Criteria.
  • List of stations:
    Clicking the blue hyper link of a station selects the corresponding station marker on the map and opens an info window with the same informations of a station in the station list. This becomes completely visible, after the view Stations was hidden.
    If a station was selected on the map, swichting to view Stations the view is scrolled to the details of this station.
Clicking the info window opens the details of the charging station with the following informations:
  • Name
  • Address
  • Connector Types with number of available charging points and power in KW
  • Operator
  • Network
  • Verified
  • Opening times
  • Location
  • Barrierfree
  • Costs with description and information of free parking and charging
  • Status: if not ok description of the fault

View Import

In view Import the html file with a station list can be selected to be imported.
With Open the selected file is shown. Clicking the blue link of a station opens the dialog for selecting the navigation system to be used to navigate to this station.

This can be used advantageously, if no internet connection is available or roaming fees should be avoided. A station list is prepared in a Wlan environment and stored at external storage. While traveling this list is imported to navigate to a selected station.

Menu items:
  • Get file: opens the dialog for selecting a station list.
  • Top: scrolls to the top of the list.
