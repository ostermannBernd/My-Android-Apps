About Cng/Lpg Finder

Cng/Lpg Finder is an Android application, which calculates a route and shows the natural gas (CNG) or liquified petroleum gas (LPG) fuel stations in Europe, USA or Canada along this route with a given maximum distance to the route or within a circle with a given radius around a location. The location may be obtained by the location service of the phone or an address entered by the user.
The fuel stations along the route can be restricted to the vicinity of refuelling stops, whose distance from each other is the cruising range with a full tank.

In Settings of the action bar the user can configure the start action, which is shown at start of the app. The start action can be the Route Request or the Circular Search of charging stations. The start action may be shown or executed immendiately applying the last used search criteria.

The following navigation systems can be used to navigate to a selected point of the route or a selected fuel station:

  • Google Maps
  • Sygic GPS
  • Navigon
  • WAZE GPS
  • HERE WeGo
  • OsmAnd
The list of stations can be saved to sd card. It can be imported to navigate to a selected station. This is done by Cng/LPG Finder without using the internet.

There is a free version Cng/Lpg Finder and a paid version Cng/Lpg Finder Plus.
Cng/Lpg Finder Plus provides additional functions for maintaining station files and searching for stations in ECO mode.
Maintaining consists of downloading station files from a web site to external storage (as sdcard), updating from external storage and resetting to the initially installed station files.
Coloured price indicators are added to station markers in map view to ease finding well priced gas stations.
In ECO mode stations for refuelling are selected so, that cost savings are a maximum compared to stations with the mean price along the route.

Setting the language

Cng/Lpg Finder is bilingual. If the language of the phone was set in 'Settings', 'Language & Keyboard' or 'Language & Input'to German, all texts of Cng/Lpg Finder appear in German, for all other languages in English.
If the language was set to English(UK) or English(US), distances are measured in miles and gas consumption is given by Miles per GB or US Gallon (Lpg) and by Miles per Kg (Cng). (Required for Eco mode).
If the language is modified, while Cng/Lpg Finder is active, then generated lists as route and station details are shown in the previous language as long as these lists are generated again.

User views

The user interacts with Cng/Lpg Finder by user views.
There are the following user views:
  • The application starts with the view Map and shows an empty map centred at the current location. After a route was calculated the route, and after gas stations are present, the stations are shown on the map.
  • View Route contains a form to request a new route or shows a formerly calculated route.
  • View Stations searches for gas stations and shows the details of gas stations.
  • Plus version only: view Maintain shows a form for downloading station data from the internet to external storage (e.g. sd card), for updating station data from external storage and for resetting data to the originally installed station data.
  • With view Import a list of stations, which was stored on external storage, can be imported.
Navigating to other views is done by option menus. The menu items are shown in the action bar on top of the screen. Dependent on available space an item is shown with its icon and possibly with title. If the space is not sufficient, the icons of the first items are shown; the remaining items are shown by clicking the vertical three dots.

Clicking the left icon (3 horizontal lines or an arrow to the left) in the action bar at the top of the screen toggles between the views Route, Map and Stations. The same can be done by swiping from the left side of the screen or back or by swiping from the right side of the screen or back.
The views Route and Stations cover a part of the map, which is slightly dimmed.

Clicking the back key immediately finishes a view or dialog and returns to the view, which initially invoked it. The data of the finished view are lost.

View Route

For requesting a route enter the address of the start location into the 'From' field and the address of the final destination into 'To' field.(Since V4.2.5 without waypoints separated by semicolon ';').

Button Waypoints opens a list of waypoints. A waypoint can be added as an address. If stations are already found, a station can be added as a waypoint by selecting it on the map and adding it in the view Waypoints or by entering the station number.
Adding a charging station as waypoint will show the loop way from the route to the station after the next route request.

By clicking the black cross on orange background a waypoint can be removed. The order of waypoints is arbitrary.
By clicking the button Ready the actual waypoints will be applied in requesting a route.
Addresses may be in the form used by Google Maps, for example:
  • complete address:Zip code, City, street house number, country name or country code
    example: 4056, Basel, Hebelstrasse 11, CH
  • incomplete address: City or City, street
  • geographical latitude, longitude in decimal degrees
    example: 50.052, 8.569
If the the From field is long clicked, the last known position of the phone is used as the From address. Google's Location Service must be activated in the phone's Settings.

Buttons:

  • Swap From/To Address:
    Start address and final destination address are swapped.
    May be applied after stations are searched in ECO Modefor the forward route and should be searched now for the backward route.
  • Avoid Highways: if checked highways will be avoided.
  • Get Route: get the route.

Menu items in the action bar: s. View Map

When the result of the route request is available the route details are shown and the route is displayed on the map.
After a successful route request the settings of the request are saved and restored after restart of Cng/Lpg Finder.

Handling of touch/click events

The route is shown as a red line together with green markers at start, stopovers and end point. The stations are presented as pushpin markers for higher zoom levels. For lower zoom levels stations are clustered; the cluster markers show the number of contained stations.
Cng/Lpg Finder Plus adds colored price indicators (green, yellow, orange, red) to station markers, which indicate the price class of a station. The price class is based on the country mean price.

Clicking a cluster marker zooms in and shows the station markers of this cluster.

Clicking a marker will open an info window with a description of this marker. A short click on the map closes the info window.

Clicking an arrow button in the middle left and right of the screen steps to the next marker, forwards by the the right button and backwards by the left button. This affects route and station markers.

Using a navigation sytem

If a route marker or a station marker is selected, clicking the button with a blue right arrow opens a dialog asking the user, if he wants to navigate with the shown navi system from the current position to the selected marker. The address of the selected marker is shown, if an internet connection exists. Clicking the field with the navi system opens a list of navi systems, from which one may be selected. Clicking Yes opens the selected navigation system, if it is installed on the android device.

The following navigation systems may be used:
  • Google Maps Navigation
  • Sygic GPS Navigation
  • Navigon with version
  • WAZE GPS Navigation
  • HERE WeGo Navigation
  • OsmAnd Navigation

If the navigation system was finished, view Map will be opened. If the android device was running on low memory, Cng/Lpg Finder may be finished and will start from scratch.

View Stations

With the view Stations Cng- or Lpg stations can be searched along a route or around a location within a given radius. The stations along a route may be restricted to the vicinity of refuelling stops, which are apart from the next stop approximately in a distance of the refuelling range. The first refuelling stop is at start of the route.
Stations along a route can be searched in ECO mode (Plus version only). In this mode stations are selected with the maximum cost savings compared to the mean price along the route.
At refuelling stops the tank is completely refilled. As a result the gas amount required for the route, the gas costs and the cost savings compared to stations with the mean price along the route are shown.
If there are no stations with pricing on a route section to the next refuelling stop, ECO mode is not applicable.

The radius search can be done around a location given by the location service of the phone or determined by an address enetered by the user.

The result of searching for stations can be saved to a html file on external storage (e.g. sd card). This file can be imported by clicking the menu item Import. This can be used for navigating to a station of this file without having an internet connection.

The station view contains two subviews:
  • Search criteria for searching stations.
  • Station list with the search results.
Each sub view may be switched to the other sub view.

Menu items: s. View Map

Search Criteria

  • Button Find stations: starts searching.
  • Button Station List: switches to sub view Station List.
  • LPG/CNG buttons: gas type of the stations to be searched for.
  • Combo box Search Mode:
    Clicking the combo box opens a list with possible search modes
    • Around Location: search around a location obtained by the location service of the phone within a circle of the given radius.
      When the search mode Around location is selected and no location service was activated, it is asked, if the settings Location access should be opened or the last known location should be used.
    • Around Address: search around a location given by the address entered by the user within a circle of the given radius.
      When this mode is already selected, the address of the location can be modified by long clicking the combo box.
      When the search mode Around address is selected and no network was activated, the settings of the telefon are opened.
    • Along Route: search along the route with a miximum distance Corridor from the route.
    • : as mode Along Route restricted to the vicinity of refueling stops. At end of the route a circular search is done to charge for the way back.
    • ECO Mode (plus version only): as mode Along Route; the refueling stations are selected in order to optimize cost savings compared to the mean price on the route.
  • Corridor: distance of the route in kms or miles for English (UK/US).
  • Input fields for Refueling Stops and ECO Mode:
    • Range with a full tank in kms or miles for language English (UK/US).
    • Remaining range in km or miles:
      Range corresponding to the actual gas level of the tank.
      The first refueling stop must occur within the remaining range.
      When searching in ECO-Modus the range after reaching the destination is shown.
    • Gas consumption (for search mode ECO mode in Plus Version):
      language different from English(UK/US): consumption in liter/100km (Lpg) or kg/100km (Cng)
      language English(UK/US): consumption in Miles per GB Gallone/US Gallone (LPG) or Miles per Kg MPkg (Cng)
The station data required for the selected search mode will be loaded, after searching for stations was started, if they are not already loaded. Only then the result of a former search will be overridden.
The states of all modifiable fields are saved after a search was completed successfully and restored after Cng/Lpg Finder was restarted.

Station List

The station list shows the stations with the following informations:
  • Name, location and gas price (not for all stations).
  • Additional informations as phone number and opening hours.
  • Distance from start of the route or from user's location and for searching along the route the airline distance from the route.
  • Geographical latitude and longitude in degrees.
  • Hyperlink Detail:
    (Only Europe, not USA and Canada)
    Clicking the link opens a web page with additional details of the station as postal address, opening hours and accepted credit cards.
  • Hyperlink Update Price
    (Only Europe, not USA and Canada)
    Clicking the link opens a dialog, which shows the actual Lpg/Cng prices, reported to the server. Clicking the Update button updates the entered new gas price on the server.
There are the following buttons or hyperlinks:
  • Button Top:
    scrolls to the top of the station list.
  • Button Save:
    Saves the station list in a html file on the external storage (e.g. sd card).
  • Button Search Criteria:
    switches to the sub view Search Criteria.
  • List of stations:
    Clicking the blue hyper link of a station selects the corresponding station marker on the map and opens an info window. This becomes completely visible, after the view Stations was hidden.
    If a station was selected on the map, swichting to view Stations the view is scrolled to the details of this station.

Update Price Dialog

Opening the Dialog shows the available actual Lpg- and Cng prices, reported to the server. Prices may be missing, if the gas station does not offer Lpg and Cng gas or the price as not yet reported.

The input field New Gas-Price contains first the actual gas price in the currency shown in the field Act. Gas-Price. The new gas price to be entered must be a decimal number with 1 to 2 digits and 1 to 3 decimal places. Clicking the button Update reports the new price to the server.
Clicking the button Get shows the new updated price.

The Update Price Dialog is only available for european gas stations, not for stations in USA and Canada.
The updated gas price is available for the Plus version after downloading the station data the next day, for the free version after installing the next version.

Note

This function evaluates an external web page. If this page will be modified by its owner, updating might not function anymore. If you experience unexpected error messages, please contact the app developer.(s. Contact)

View Import

In view Import the html file with a station list can be selected to be imported.
With Open the selected file is shown. Clicking the blue link of a station opens the dialog for selecting the navigation system to be used to navigate to this station.

This can be used advantageously, if no internet connection is available or roaming fees should be avoided. A station list is prepared in a Wlan environment and stored at external storage. While traveling this list is imported to navigate to a selected station.

Menu items:
  • Get file: opens the dialog for selecting a station list.
  • Top: scrolls to the top of the list.

View Maintain

(Available in Cng/lpg Finder Plus only)

View Maintain lets the user maintain the gas station data.
A gas station file contains informations about the gas stations of a country.

( s. Supported Countries)

These informations consist of location, name and optional additions as price, opening hours and payment mode (cash, credit card), which often are updated, in particular prices.
The station data come from a web site, which updates the files each night.
Cng/Lpg Finder comes with an initial set of gas station data, which become the actual data, after Cng/Lpg Finder is first started after installation.
Maintaining gas station data means:
  • Downloading currency rates:

    Downloading currency rates from the European Central Bank ECB. The rates are used for converting prices in non Euro countries to Euros.

  • Downloading station data:

    The station data are downloaded from a web site to external storage (as sdcard). Downloading is done by a background service, which runs independently on the app Cng/Lpg Finder Plus. The end of download is notified in the status bar at the top of the screen. Swiping down will show the results of the downlaod. After downloading was finished successfully, station data will be updated from sdcard.

    The download service can be started explicitly by the user or can run periodicly. If the periodic mode is selected, at start of the app the service is started, when the period has elepased after the last download.

    Settings:

    Settings are done in the view Download

    Selection of data types (with multiple selection):
    • Cng Eur: Cng data Europe
    • Lpg Eur: Lpg data Europe
    • Cng USA: Cng data USA
    • Cng CAN: Cng data Canada
    • Lpg USA: Lpg data USA
    • Lpg CAN: Lpg data Canada

    Periodic downloading:

    Checking the checkbox Periodic activates the periodic mode. The period in weeks is set in the field Weeks.

    Explicit start:

    Clicking the button Start will start the download service; the checkbox Periodic must be unchecked.
  • Updating station data from sdcard:

    If the external storage contains station data in a specific directory structure, clicking the button 'update station data ..' will replace the actual station data by those on sdcard.

  • Resetting to initial station data:

    If it turns out that the updated gas data are incorrect, the user can reset to the initially installed station data.

Note:

The author of Cng/Lpg Finder takes no warranty that the downloaded files are correct, and takes no responsibility about damages resulting by using incorrect gas files.

Version V1.1.0

New functions:

  • Search for stations in ECO mode (Plus version):
    The refueling stations along the route are selected in such a way, that cost savings are a maximum compared with the mean gas price along the route.
  • Distances in miles for English(UK):
    If the language English(UK) is selected, distances are given in miles rather than kilometers.
  • Save and restore settings:
    The settings of views Route and of Stations are saved, after a successful route request or search for stations. The saved settings are restored after restarting Cng/Lpg Finder.
  • Gas price additionally in Euros:
    The gas price in station details and the info window of a station is added in Euros, if the local currency is not Euro.
  • Currency rates are downloaded from the European Central Bank ECB, when the station files are downloaded (Plus version only).
  • Moving application to sdcard and vice versa:
    If the smartphone runs Android 2.2 or higher, Cng/Lpg Finder may be moved from internal memory to sdcard or vice versa (In Settings, Manage Applications).

Bug fixes:

  • Out of memory error when zooming:
    After a large number of stations were found zooming the map may result in an out of memory error.
    The error was caught and the application falls back to requesting the route. The maximum number of stations was reduced to 600.
